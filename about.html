Free Food @ RU

About

RuFreeFood connects hungry college students with events that offer free food.
For each free food event the website also displays name, date, time, location, type of food offered, and description.

Features:

  • You can add an event to your Google/Yahoo/ICalendar by clicking on the calendar icon.
  • View a map of all the events in the 'Map View' tab.
  • You can visit the original source by clicking the title of the event.
  • Posting an event that has free food. You can do so by going to 'Posts' tab and clicking the 'Add Post' button.
If you would like to see new features to be implemented than please contact by using the 'Contact Us' link at the bottom.

How it works

There's a script that runs every day and scrapes information from couple of Rutgers Events Websites and adds all the event information into a database.
When you visit rufreefood.club the website gets the data from the database, checks for events that are more likely to contain free food, and displays the information in a well-orgainzed manner.
